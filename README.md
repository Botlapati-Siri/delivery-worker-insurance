# Delivery Worker Safety Platform

## Problem Statement
Delivery workers face financial risk due to accidents and unpredictable income. 
Our platform provides affordable weekly insurance protection for gig workers.

## Objective
Build a parametric insurance system that automatically detects accidents and provides quick payouts.

## Target Users
Food delivery workers from platforms like Swiggy and Zomato.

## Key Features
- Weekly premium model
- Automatic accident detection
- AI-based risk assessment
- Fraud detection
- Quick insurance claim processing

## Platform
Mobile application for Android devices.
